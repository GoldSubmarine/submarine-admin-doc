# 代码生成

本项目使用 xcrud-generator 实现代码生成，具体用法点击链接查看文档

查询数据库表字段，快速生成 ejs 模板代码，不区分项目语言，前后端都可使用

优点

  • 无侵入，只需在项目中添加 json 配置文件，写好模板即可
  • 自定义参数，根据自身需求，在页面中混入参数
  • 动态路径，生成的文件路径可动态生成，支持 ejs 语法
